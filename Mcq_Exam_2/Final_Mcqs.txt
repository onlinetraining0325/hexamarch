Here are important MCQs on Spring REST (Spring Boot) using Spring Boot — useful for interviews & exams.

🔹 Spring REST MCQs
✅ 1. Which annotation is used to create REST APIs in Spring?

A) @Controller
B) @RestController
C) @Component
D) @Service

👉 Answer: B) @RestController

✅ 2. What does @RestController combine?

A) @Controller + @Service
B) @Controller + @Repository
C) @Controller + @ResponseBody
D) @Component + @ResponseBody

👉 Answer: C) @Controller + @ResponseBody

✅ 3. Which annotation maps HTTP GET requests?

A) @PostMapping
B) @PutMapping
C) @GetMapping
D) @RequestBody

👉 Answer: C) @GetMapping

✅ 4. Which annotation is used to read JSON input?

A) @PathVariable
B) @RequestBody
C) @RequestParam
D) @ResponseBody

👉 Answer: B) @RequestBody

✅ 5. Which annotation is used for URL parameters?

A) @RequestBody
B) @PathVariable
C) @Autowired
D) @Component

👉 Answer: B) @PathVariable

✅ 6. Default return type of REST API is?

A) HTML
B) XML
C) JSON
D) Text

👉 Answer: C) JSON

✅ 7. Which HTTP method is used to update data?

A) GET
B) POST
C) PUT
D) DELETE

👉 Answer: C) PUT

✅ 8. Which HTTP method is idempotent?

A) POST
B) GET
C) PUT
D) Both B & C

👉 Answer: D) Both B & C

✅ 9. Which annotation handles query parameters?

A) @RequestParam
B) @PathVariable
C) @RequestBody
D) @Autowired

👉 Answer: A) @RequestParam

✅ 10. Which class is used to customize HTTP response?

A) HttpServlet
B) ResponseEntity
C) RestTemplate
D) DispatcherServlet

👉 Answer: B) ResponseEntity

✅ 11. Which annotation is used for exception handling globally?

A) @ExceptionHandler
B) @ControllerAdvice
C) @RestControllerAdvice
D) All of the above

👉 Answer: D) All of the above

✅ 12. Which dependency is required for REST APIs?

A) spring-core
B) spring-context
C) spring-web
D) spring-aop

👉 Answer: C) spring-web

✅ 13. What is the default port of Spring Boot?

A) 8081
B) 8080
C) 9090
D) 80

👉 Answer: B) 8080

✅ 14. Which annotation is used to map base URL?

A) @GetMapping
B) @RequestMapping
C) @PostMapping
D) @Component

👉 Answer: B) @RequestMapping

✅ 15. Which tool is commonly used to test REST APIs?

A) Eclipse
B) IntelliJ
C) Postman
D) Maven

👉 Answer: C) Postman

🔥 Advanced MCQs
✅ 16. What does @RequestMapping(method = RequestMethod.GET) equal to?

B) 201
